## OGP画像テキスト

noteのサムネイル画像に使うテキスト:

```
【実態調査】
みんなの家賃
本当はいくら？

XXX人の回答データ
徹底分析

✓ エリア別の相場
✓ 年代別の家賃比率
✓ 満足度と家賃の相関
```

画像仕様:
- サイズ: 1280×670px（noteの推奨サイズ）
- 背景: 濃紺グラデーション(#1a2744 → #667eea)
- テキスト: 白
- Canvaで作成推奨

## note投稿手順

1. [ ] https://note.com にログイン
2. [ ] 「投稿」→「テキスト」を選択
3. [ ] タイトル入力: `【実態調査】みんなの家賃、本当はいくら？XXX人の回答データを分析`
4. [ ] 本文をMarkdownからnote形式にペースト
5. [ ] 有料ラインを「ここから先は有料パートです」の直前に設定
6. [ ] 価格設定: ¥500
7. [ ] サムネイル画像をアップロード
8. [ ] ハッシュタグ: #家賃 #一人暮らし #家計管理 #節約 #不動産 #お金の勉強
9. [ ] プレビューで確認
10. [ ] 公開
