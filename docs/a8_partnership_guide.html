A8.net 提携申請ガイド

内部用マニュアル ― 各アフィリエイトプログラムへの提携申請手順

前提条件 A8.net にログイン済みであること
ログインID: freemlife
管理画面: https://pub.a8.net/

共通手順

  1. A8.net にログインする
  2. 「プログラム検索」画面を開く
  3. キーワード欄に検索語を入力する
  4. 「検索」ボタンをクリック
  5. 該当プログラムの「提携申請する」をクリック
  6. 確認ページで再度「提携申請する」をクリック

結果: 「提携完了」（即時提携）または「審査中」と表示される

提携申請後の対応

  • 即時提携の場合: すぐに広告リンクが作成可能です。「広告リンク作成」画面へ進んでください。
  • 審査ありの場合: 数日〜1週間で結果がメール通知されます。承認後に広告リンクを作成してください。
  • リンク取得後: config/affiliate_links.json の該当キーの url フィールドを、取得したアフィリエイトリンクで更新してください。
  • ツールへの反映: 以下のコマンドを実行して、各ツールのHTMLにリンクを注入します。
python scripts/inject_affiliate_links.py
